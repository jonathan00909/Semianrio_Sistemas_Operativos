Simulación de Planificación de Procesos
Este proyecto es una simulación de varios algoritmos de planificación de procesos implementados en Python 3.10.9. Los algoritmos incluidos son Round Robin, Shortest Job First (SJF), First In, First Out (FIFO) y Prioridades.

Requisitos
Asegúrate de tener Python 3.10.9 o una versión superior instalada en tu sistema. Puedes verificar tu versión de Python ejecutando el siguiente comando en tu terminal: python --version

Para este proyecto, no es necesario instalar bibliotecas externas adicionales, ya que solo se utilizan las bibliotecas estándar de Python.

Uso
Coloca un archivo de texto llamado "procesos.txt" en la misma ubicación que el script de Python. El archivo debe contener una lista de procesos en el siguiente formato:

Nombre_Proceso, Tiempo_Ejecucion, Prioridad
Ejemplo:
Proceso_A, 5, 2
Proceso_B, 3, 1
Proceso_C, 6, 3


Asegurate de que pongas la dirección del archivo procesos en "ruta_archivo" como en mi caso es: r'C:\Users\Usuarios\OneDrive\Escritorio\Nueva carpeta (2)\procesos.txt'
